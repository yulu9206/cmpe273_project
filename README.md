# Baby Food Chain
```
Implemented a simple blockchain in Python using Flask.
```
### Functional Requirement
```
Use blockchains to register baby foods: Formula, snacks, etc.
```
### How to run
```
python3 app.py
python3 app.py -p 5001
```
